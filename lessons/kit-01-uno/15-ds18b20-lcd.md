# Lesson 15 — DS18B20 temperature sensor with I2C LCD

**Source:** [TENSTAR ROBOT - RFID Starter Kit (Arduino UNO R3)](https://pl.aliexpress.com/item/1005005970564532.html)

## Goal

Read temperature from a DS18B20 1-Wire sensor, display °C on the I2C LCD, and turn on an LED when temperature exceeds 30 °C.

## Materials

| Qty | name | `parts.yaml` id |
|-----|------|-----------------|
| 1 | Arduino UNO board | `arduino-uno-r3` |
| 1 | DS18B20 temperature sensor | `ds18b20` |
| 1 | LCD 1602 with I2C backpack | `lcd1602-i2c` |
| 1 | LED 5 mm + resistor | `led-5mm`, `resistor-220` |
| 1 | Breadboard | `breadboard` |
| — | Jumper wires | `jumper-wires` |

## Key concepts

- 1-Wire bus (single data pin + 4.7 kΩ pull-up)
- `DallasTemperature` / `OneWire` libraries
- Threshold alarm with digital output

## Libraries

- `OneWire`
- `DallasTemperature`
- `LiquidCrystal_I2C`
