# Lesson 8 — 74HC595 and 1-digit 7-segment display

**Source:** [TENSTAR ROBOT - RFID Starter Kit (Arduino UNO R3)](https://pl.aliexpress.com/item/1005005970564532.html)

## Goal

Display digits 0–9 on a single common-cathode 7-segment display driven through a 74HC595 shift register.

## Materials

| Qty | name | `parts.yaml` id |
|-----|------|-----------------|
| 1 | Arduino UNO board | `arduino-uno-r3` |
| 1 | 74HC595 shift register | `74hc595` |
| 1 | 1-digit 7-segment display | `7seg-1digit` |
| 1 | Resistor 220 Ω | `resistor-220` |
| 1 | Breadboard | `breadboard` |
| — | Jumper wires | `jumper-wires` |

## Previous knowledge

## Key concepts

- 7-segment segment map (A–G + decimal point)
- Common cathode vs common anode
- Byte patterns per digit via `shiftOut()`
