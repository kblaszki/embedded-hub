# Displays and input

See [parts.yaml](parts.yaml) → `displays` and `modules`. Kits: [kit-01-uno](kits/kit-01-uno.md), [kit-04-esp32](kits/kit-04-esp32.md).

| id | name | interface |
|----|------|-----------|
| lcd1602-i2c | LCD 1602 I2C | i2c |
| lcd128x32-i2c | LCD / OLED 128×32 I2C | i2c |
| led-matrix-8x8 | 8×8 LED matrix MAX7219 | spi |
| 7seg-4digit | 4-digit 7-segment | gpio |
| 7seg-1digit | 1-digit 7-segment | gpio |
| keypad-4x4 | 4×4 matrix keypad | gpio |

RFID (`rfid-rc522`, fob, card) is under **modules**.
