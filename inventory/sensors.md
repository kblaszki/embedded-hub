# Sensors

Modules and bare sensors. See [parts.yaml](parts.yaml) → `sensors`.

Currently empty — add items as you acquire hardware.

## Suggested categories

| Type | Examples | Typical interface |
|------|----------|-------------------|
| Environment | DHT11, DHT22, BME280 | digital / I2C |
| Distance | HC-SR04, VL53L0X | digital / I2C |
| Light | Photoresistor, BH1750 | analog / I2C |
| Motion | PIR HC-SR501 | digital |
| Input | Rotary encoder | digital |

Copy a block from the commented examples in `parts.yaml` when adding a new sensor.
